# MYPORTFOLIO 🚀

Welcome to the source code of my professional software engineering portfolio. This repository contains a premium, ultra-modern, and highly responsive personal website built to showcase my full-stack development projects, UI/UX designs, and technical expertise.

## ✨ Key Features

- **Premium Glassmorphism Aesthetic:** A deep dark-mode design system utilizing sleek surface containers, neon primary accents, and subtle outline borders.
- **Intersection Observer Animations:** High-performance, buttery-smooth scroll animations that gracefully slide and fade elements into view as you navigate the page.
- **Interactive Project Carousel:** A custom-built, mobile-responsive horizontal snapping carousel to showcase featured case studies with uncropped, perfectly framed UI mockups.
- **Fully Responsive:** Fluid layouts built with modern CSS Grid and Flexbox, ensuring perfect pixel alignment from 4K desktop monitors down to mobile screens.

The portfolio highlights several major full-stack and SaaS applications:

2. **Enterprise Databricks Portal:** A robust dashboard for real-time big data visualization and cloud resource orchestration (React, Azure, D3.js).

## 🛠️ Technology Stack

- **Structure:** HTML5 Semantic Elements
- **Styling:** Custom CSS leveraging modern Tailwind-inspired utility classes (e.g., `bg-surface-container`, `text-on-surface-variant`) and CSS Custom Properties for theme tokens.
- **Interactivity:** Vanilla JavaScript (ES6+)
- **Animations:** CSS Transitions & Intersection Observer API
- **Icons:** Google Material Symbols (Outlined)
- **Typography:** Google Fonts (Outfit, Inter, Space Mono)

## 🚀 Local Development

To view the portfolio locally, you simply need to serve the HTML files. No build step is required!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nachu46/MYPORTFOLIO.git
   cd MYPORTFOLIO
   ```

2. **Serve the directory:**
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   Or using Node.js (`serve`):
   ```bash
   npx serve .
   ```

3. **Open your browser:**
   Navigate to `http://localhost:8000` to view the site.

## 📁 Repository Structure

- `index.html`: The main landing page featuring the hero section, skills, timeline, and featured project carousel.
- `demo.html`: The comprehensive directory of all additional web applications and tools.
- `/demo-*`: Individual directories containing deep-dive case studies for specific applications.
- `/assets/`: High-resolution AI-generated UI mockups and graphics used throughout the site.
- `/css/`: Core stylesheet definitions and design system variables.
