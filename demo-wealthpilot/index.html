WEALTHPILOT AI

A cutting-edge web application designed to deliver real-time stock/market predictions, AI-driven portfolio insights, and automated trading analysis using Next.js and FastAPI.

Project Architecture

WealthPilot AI features a modern Next.js frontend integrated with a powerful, scalable FastAPI backend. The project is designed with a strict micro-architecture, splitting the React UI components and Python API endpoints for ultimate scalability.

🐋 Running with Docker

To spin up the entire multi-container stack (frontend, backend, database) with a single command:

docker-compose up --build

1. Local Backend Setup (FastAPI)

# Navigate to backend cd backend # Activate virtual environment # Windows: venv\Scripts\activate # macOS/Linux: source venv/bin/activate # Install dependencies pip install -r requirements.txt # Run database migrations alembic upgrade head

Start the FastAPI server (runs on http://127.0.0.1:8000):
uvicorn main:app --reload

2. Local Frontend Setup (Next.js)

# Navigate to frontend cd ../frontend # Install dependencies npm install # Start development server npm run dev

The frontend will be running at http://localhost:3000.
