About the Platform

A complete visual workflow builder with real integrations and code generation capabilities. Built on top of Workflow DevKit, this template provides everything you need to build your own AI-driven workflow automation platform with Node.js and React.

terminal

Code Generation

Workflows can be instantly converted to executable, type-safe TypeScript code with the "use workflow" directive. The generated code includes real integration calls, error handling, and execution logging.

check_circle Fully Executable TypeScript

bolt Real Integrations & AI Powered

Connect to external APIs seamlessly and generate entire workflows from natural language descriptions.

Native Integrations

Built-in support for Resend (emails), Linear (tickets), Slack, Stripe, PostgreSQL, and Firecrawl scraping.

AI Workflow Generation

Powered by OpenAI, simply type what you want your workflow to do and the AI will generate the visual nodes automatically.

Local Development Setup

See More Hide expand_more

1. Prerequisites & Environment

Ensure you have Node.js 18+, pnpm, and PostgreSQL. Create a .env.local file:

DATABASE_URL=postgresql://user:password@localhost:5432/workflow_builder BETTER_AUTH_SECRET=your-secret-key BETTER_AUTH_URL=http://localhost:3000 AI_GATEWAY_API_KEY=your-openai-api-key

2. Installation & Database

# Install dependencies pnpm install # Run database migrations pnpm db:push

3. Development Server

# Start development server pnpm dev

Visit http://localhost:3000 to get started.
